================================================================================
                         ARCHITECTURE DESIGN
================================================================================

LAYERED ARCHITECTURE:

Request
  ↓
Express App (app.js)
  ↓
Routes (transcriptionRoutes.js)
  ↓
Middleware - File Validation (uploadMiddleware.js)
  ↓
Controller - Business Logic (transcriptionController.js)
  ↓
Service - Whisper API (whisperService.js)
  ↓
Config - OpenAI Client (openai.js)
  ↓
OpenAI Whisper API
  ↓
Response JSON
  ↓
Frontend

BENEFITS:
  ✓ Easy to test (each layer isolated)
  ✓ Easy to extend (add features without refactoring)
  ✓ Easy to maintain (clear responsibilities)
  ✓ Easy to scale (separate concerns)

================================================================================
                        TROUBLESHOOTING
================================================================================

PORT 5000 ALREADY IN USE:
  Solution: Change PORT in .env or kill process using port 5000

"Cannot find module" error:
  Solution: Run "npm install" in Backend folder

"Invalid API Key" error:
  Solution: Check OPENAI_API_KEY in .env file

"File type not allowed" error:
  Solution: Ensure file is audio (webm, wav, mp3, mpeg)

"File too large" error:
  Solution: File size > 25MB, record shorter audio

CORS Error (frontend):
  Solution: CORS already enabled, check frontend URL matches
