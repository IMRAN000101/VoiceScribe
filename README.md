# VoiceScribe AI

VoiceScribe AI is an AI-powered speech-to-text platform built with React, Express.js, and OpenAI Whisper.

Features:
- Real-time audio recording using MediaRecorder
- Microphone access through Web APIs
- Audio upload pipeline using FormData and Express
- Speech-to-text transcription with OpenAI Whisper
- Modern responsive SaaS UI
- Dark/Light theme support

Upcoming Features:
- AI Summaries
- Key Points Extraction
- Action Items Detection
- User Authentication
- Transcript History
- PDF Export

Tech Stack:
Frontend: React, Tailwind CSS, React Router
Backend: Node.js, Express.js, Multer
AI: OpenAI Whisper API
